üìö Step-by-Step Learning

Learn how to solve problems step by step!

1

Addition Steps

  • Write numbers in columns
  • Start from ones place
  • Add digit by digit
  • Carry over if sum > 9

Example: 1,37,462 + 4,005 + 38 + 56,734

2

Subtraction Steps

  • Write larger number on top
  • Start from ones place
  • Borrow if needed
  • Subtract digit by digit

Example: 85,000 - 67,432

3

Multiplication Steps

  • Multiply by ones digit
  • Multiply by tens digit
  • Add the partial products
  • Place values correctly

Example: 35 √ó 350

4

Division Steps

  • Divide from left to right
  • Find how many times divisor fits
  • Multiply and subtract
  • Bring down next digit

Example: 53,675 √∑ 8
