🎯 Easy Management Rules

✅ What Makes This Easy:

  • Consistent Structure: Every standard follows the same pattern
  • Simple Links: Just use standard number + subject name
  • Easy to Copy: Copy std1 structure for std2, std3, etc.
  • Clear Organization: Each subject has its own folder
  • Flexible Content: Can have lessons.html or index.html

📝 Quick Setup Guide

🚀 For New Standards

  1. Copy std1 folder → std2
  2. Rename std1.html → std2.html
  3. Update navigation links
  4. Update breadcrumbs
  5. Update titles and descriptions

📚 For New Subjects

  1. Create subject folder (e.g., maths/)
  2. Add index.html (Coming Soon page)
  3. Or add lessons.html + lessons.json (if has content)
  4. Update navigation mapping in std*.html

🔧 Navigation Code Template

// Standard navigation in each std*.html file: document.querySelectorAll('.class-card').forEach(card => { card.addEventListener('click', (e) => { const subject = card.dataset.subject; // Simple mapping - easy to maintain const subjectMap = { 'tamil': 'tamil', 'english': 'english', 'maths_en': 'maths', 'science_en': 'science', 'social_en': 'social', 'maths_ta': 'maths', 'science_ta': 'science', 'social_ta': 'social' }; const folderName = subjectMap[subject]; if (folderName) { // Check if subject has content or is coming soon if (subject === 'tamil') { window.location.href = `${folderName}/lessons.html`; } else { window.location.href = `${folderName}/index.html`; } } }); });
